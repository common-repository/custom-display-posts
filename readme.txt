Adds a widget that shows the posts in a single or more category.


== Description ==

This plugin allows you to easily custom display posts on your site.

= Features =
*  Supports single or more category display
*  Support show the post comments count
*  Support order by Posts today, yesterday, 7 days, 1 month, 3 months, 6 months, 12 months 

== Installation ==

1. Unzip & upload the plugin directory inside your /wp-content/plugins/ directory 
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. Go to 'Widgets' and configure
